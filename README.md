# iclean-Frontend


## Development Environment Setup


  1. Clone the repo using the below command

      https://github.com/khnehal/iclean-Frontend.git

  2. Move to `iclean-frontend` folder and install `npm` dependencies.
  
      npm install 
          
          or 

      yarn install

  3. Start the server using

      npm start
      
          or

      yarn start
